# opencv-js-demo

This is a demo project to try OpenCV.js

Latest version is https://docs.opencv.org/4.5.3/opencv.js

You can find it from the doc: https://docs.opencv.org/4.5.3/d3/de5/tutorial_js_houghcircles.html


## Run

```
yarn serve
```
